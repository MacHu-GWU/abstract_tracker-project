Install
------------------------------------------------------------------------------

``abstract_tracker`` is released on PyPI, so all you need is to:

.. code-block:: console

    $ pip install abstract-tracker

To upgrade to latest version:

.. code-block:: console

    $ pip install --upgrade abstract-tracker
